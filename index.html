HTML

  • Head Elements contains information about the webpage.
  • Body Elements represent the visible content shown to the user.
  • Section Elements split the structure into organized portions.
  • Classes can be assigned to elements as a way to specifically style them with CSS.
  • You can use ul (unordered lists) coupled with li (list items) to create lists that will be rendered in the webpage.

CSS

  • Margin indicates how much space we want around the outside of an element.
  • Padding indicates how much space we want around the content inside an element.
  • Placing a period before a name in CSS references an object class instead of an element.
  • Colors can be assigned with both semantic reference and hexadecimal values. use this site to find hexadecimal color codes.
  • Make sure that all declerations end with a semicolo ;

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages the changed files for deposition.
  • git commit -m "message": adds a commit message to the new file entry.
  • git pull origin main: downloads content from the main repository branch, and updates the local repository to match.
  • git push origin feature/example: pushes the new files into the branch.
  • cd command: allows you to navigate into different folders and files.
  • code .: command allows you to open a folder or file in the Visual Studio Code program.

JavaScript

  • Variables: are a named container that allows you to store data within the script.
  • Control flow is the order in which a computer executes code in a script.
  • Functions allow us to store complex systems under a name that can be called apon later in the script.
  • Make sure that all lines of code end with a semicolo ;
